Link Github: https://github.com/TNHao/Reversi

* Cách cài đặt (sử dụng terminal):

- Cho client:
    + Di chuyển đến thư mục client:
        cd client
    + Cài đặt package cho client:
        npm install

- Cho server:
    + Di chuyển đến thư mục server:
        cd client
    + Cài đặt package cho server:
        npm install


* Chạy ứng dụng (sử dụng terminal):

- Chạy client:
    npm start

- Chạy server:
    npm start
